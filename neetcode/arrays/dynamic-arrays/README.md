# Time Complexity

| Operation | Time Complexity | Notes                                                           |
| --------- | --------------- | --------------------------------------------------------------- |
| Access    | O(1)            |                                                                 |
| Insertion | O(1)            | O(n) if insertion in the middle since shifting will be required |
| Deletion  | O(1)            | O(n) if deletion in the middle since shifting will be required  |
