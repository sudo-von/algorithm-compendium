# Time Complexity

| Operation | Time Complexity | Notes                                      |
| --------- | --------------- | ------------------------------------------ |
| Reading   | O(1)            |                                            |
| Insertion | O(n)            | If inserting at the end of the array: O(1) |
| Deletion  | O(n)            | If deleting at the end of the array: O(1)  |
